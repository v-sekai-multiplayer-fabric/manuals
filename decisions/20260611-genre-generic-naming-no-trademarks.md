## Context and Problem Statement

The core loop draws on a trademarked action RPG. The project reuses none of that game's trademarks, so the docs and the game need a neutral vocabulary that carries the mechanics without the marks.

## Decision Outcome

Chosen option: use genre-generic vocabulary throughout the docs, the filenames, and the game, because generic terms describe the mechanics while carrying no trademark.

The vocabulary: Hub for the social staging area, Field for a combat instance, melee, ranged, and caster for the archetypes, art for a learnable ability, affinity for the gating stat, and energy for the ability resource.

## Consequences

- No trademarked proper nouns appear in filenames, docs, or content.
- The docs read as their own product.
- New writing reuses the same neutral terms, so the vocabulary stays consistent.

## Confirmation

A trademark scan over the project docs returns clean.
